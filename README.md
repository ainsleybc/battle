# Battle
Makers Academy Intro to web week - a battle game modelled after Pokemon

## Technologies Used

Languages - Ruby, HTML & CSS  
Framework - Sinatra  
Testing - RSpec & Capybara  

## Usage

```
bundle install
```

visit `http://localhost:9292/` in your browser

## Test

```
rspec
```
